●概要

    mp3等の形式の音声ファイルをWAV形式に変換するコマンドラインツールです。
    変換するときに任意のビットレートを指定できます。


●特徴

    ・  コマンドライン形式なのでバッチ処理が可能
    ・  任意のビットレートに変換できる
    ・  WAVファイル同士の変換も可能
    ・  Windows Media Playerで再生可能なファイルであれば変換可能
    ・  動画ファイルから音声の抽出も可能


●確認済みフォーマット

    ・  .wav
    ・  .mp3
    ・  .ra (Real Audio) ※Real Playerのインストールが必要
    ・  .avi
    ・  .mpg
    ・  .wmv


●動作環境

    Windows 2000/XP/Vista (Vistaは動作未確認) 
    Windows Me/98はたぶん対応 (動作未確認)

●インストール/アンインストール

    【インストール方法】
    ダウンロードした圧縮ファイルを任意のインストールしたいフォルダに
    解凍してください。
    
    ※Windows XP/Vistaの場合は以上でインストール完了です。
    ※Windows Me/98/2000の場合は以下を実行してください。

    コマンドラインからインストールしたフォルダに移動し、
    『regsvr32 wavdest.ax』を実行してください。
    「wavdest.axのDllResisterServerは成功しました。」というメッセージ
    ボックスが表示されたら[OK]を押せばインストール完了です。

    【アンインストール方法】
    ※インストール時に『regsvr32 wavdest.ax』を実行していない場合は、
    フォルダを削除するだけでアンインストール完了です。

    アンインストールするときはコマンドラインからインストールした
    フォルダに移動し、『regsvr32 /u wavdest.ax』を実行し、
    「wavdest.axのDllUnResisterServerは成功しました。」という
    メッセージボックスが表示されたら[OK]を押してください。
    その後、フォルダを削除すればアンインストール完了です。


●使用方法

    コマンドラインで変換するファイルを引数にしてwavextを実行する。
    ビットレートを指定するときは第1,2,3引数にそれぞれ、
    ステレオ(2)/モノラル(1)、サンプルレート、サンプルビットを指定する。
    変換後のファイルは元のファイルと同じフォルダに出力される。

    引数なしで実行すると引数の説明が表示される。

    usage : wavext [Channels SamplesPerSec BitsPerSample] <file> [<output file>]
        Channels        1:mono 2:stereo
        SamplesPerSec   8000 11025 16000 22050 32000 44100 etc
        BitsPerSample   8 16
    
        example)   wavext test.mp3
                   wavext 2 44100 16 test.mp3
                   wavext test.mp3 out.wav
                   wavext 2 44100 16 test.mp3 out.wav
